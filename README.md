
##  Cybersecurity Projects:

- [Wardriving Rig with Raspberry Pi 4](https://github.com/dustinGodfrey/wardriver)
    - A fully headless, portable wardriving rig built with a Raspberry Pi 4, Kismet, GPS, and custom Python scripts — featuring LED/button hardware controls, automated logging, and a locally-hosted interactive WiFi network map generated with Pandas and Folium. The project revealed real-world wireless security gaps in the wild, including 16% of discovered networks using default manufacturer credentials.

- [AAA Labs](https://github.com/dustinGodfrey/AAA-Labs)
    - Hands-on homelab documentation covering enterprise authentication protocols including PKI with OpenSSL, 802.1X network access control, and FreeRADIUS configuration with EAP-TTLS, PEAP, and EAP-TLS. Labs were built in a Proxmox virtualization environment and analyzed at the packet level using Wireshark.

- [BHP Labs](https://github.com/dustinGodfrey/Black-Hat-Python)
    - A chapter-by-chapter working repository following Black Hat Python by Justin Seitz and Tim Arnold, building offensive security tools from scratch in Python to deepen understanding of networking, reconnaissance, and penetration testing concepts.

- [Bandit - OverTheWire Walkthrough](https://github.com/dustinGodfrey/Bandit-OverTheWire)
    - A documented personal walkthrough of OverTheWire's Bandit wargame, detailing the methodology, Linux commands, and problem-solving approach used at each level to strengthen command-line, file system navigation, and reconnaissance skills.
